# Handoff template (mandatory shape)

Copy into `NEXT_SESSION_HANDOFF.md` on every **`fin`**. **Nothing else.**

```markdown
# <Project> — HANDOFF

| | |
|---|---|
| **fait** | |
| **commit** | `` |
| **P0** | |

*YYYY-MM-DD*
```

**Forbidden in handoff:** chat recap, file lists, duplicated spec/brief, « State » sections, starter-block text.
